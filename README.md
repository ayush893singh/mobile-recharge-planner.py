# Mobile Recharge Planner

A simple Python project that allows users to select a mobile recharge plan and displays the recharge details, including the selected plan and its validity.

---

Features

- Enter a mobile number
- Display multiple recharge plans
- Select a recharge plan
- Show recharge amount
- Show plan validity
- Beginner-friendly Python project

---

Technologies Used

- Python

---

Concepts Used

This project uses:

- Variables
- User Input
- Conditional Statements (if, elif, else)
- Output Formatting
- Basic Python Syntax

---

Available Recharge Plans

Plan| Price| Validity
Plan 1| ₹199| 28 Days
Plan 2| ₹299| 28 Days
Plan 3| ₹399| 56 Days
Plan 4| ₹599| 84 Days

---

Installation

1. Install Python on your computer.
2. Download or clone this repository.
3. Open a terminal in the project folder.

---

How to Run

python main.py

---

How It Works

1. Enter your mobile number.
2. View the available recharge plans.
3. Select a plan by entering its number.
4. The program displays the recharge summary.
